## 🔧 Refactoring
- position + left 좌표 기반 정렬 → left: 50% + transform 구조로 개선  
- 고정 width 제거 → width: 100% 기반 레이아웃으로 수정  
- 검색창 레이어 1920px 고정값 제거 및 full-width 처리  
- 드롭다운 메뉴 구조 및 콘텐츠 원본 기준으로 재정리

## 💡 What I Learned
- position과 고정 좌표(px, %) 기반 레이아웃이 다양한 해상도에서 깨지는 문제를 경험  
- left 기준 정렬이 아닌, transform을 활용한 중앙 정렬 방식의 중요성 이해  
- 고정 width(1920px) 사용이 아닌, 유동적인 레이아웃 설계의 필요성을 체감
- UI 요소를 개별 위치가 아닌, 전체 레이아웃 흐름 안에서 설계하는 방식 학습
